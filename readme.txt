A video Embed Plugin that allows to embed videos from Youtube, Vimeo, Wistia or even Mp4 videos with some advanced options and creates shortcodes.

== Description ==

This plugin makes it very simple and easy for users to upload videos to their posts and pages. It will let you select your video, choose your settings and save those settings, it will create a shortcode for you that you can include in any of your posts/pages. 

Video Types Available:
1) Youtube
2) Vimeo
3) Wistia
4) Mp4 Upload
5) Mp4 URL 

This plugin offers much more than just embedding the videos. It offers really cool features for video marketers. Following are some features that proves this plugin is different from the other video plugins available:
1) Video box floating on left/right side on browser scroll, so your website users can visit the whole page and at the same time can watch the video.
2) It allows you to add Call-to-action box at bottom of the video, where you can add link to you website or product.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/hgt-video-embed` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to your website\'s Admin Dashboard and look for 'Video Shortcodes' tab.
4. Create New shortcode and add it to your Posts/Pages.
